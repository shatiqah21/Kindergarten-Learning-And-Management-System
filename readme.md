Pasti Firdaus Kindergarten Learning & Management System

This system is designed to streamline the management of kindergarten operations, including student records, teacher management, class management, learning materials, and performance assessments. It enables teachers and administrators to handle information more efficiently and systematically, improving productivity and the quality of educational management.

Key Features:

Student registration and management
Teacher and class management
Learning materials management
Student performance and development assessment
Digital reports and record updates

Technologies Used:
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Development Platform: XAMPP & VS Code

Project Objective:
This FYP aims to develop a user-friendly and efficient learning and management system for kindergartens, helping teachers and administrators track student progress easily.
